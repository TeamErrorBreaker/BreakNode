우리가 정말 구현하고자 하는 기능들은 모두 비동기에 의해 구현해야 했고, 이 부분을 명확히 해결하려면 통신 프로토콜에 대한 이해와 이를 활용할 수 있는 Front-End 영역의 지식도 필요하다고 느꼈다.

Full-Stack 개발자로 향하는 것을 목표로 한다!
